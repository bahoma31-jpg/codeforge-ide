# وضع القراءة فقط - Read Only Mode

## القواعد الصارمة:

1. **ممنوع الحذف:** لا تحذف أي ملف أو مجلد تحت أي ظرف
2. **ممنوع التعديل:** لا تعدل أي كود موجود في أي ملف
3. **ممنوع الإضافة:** لا تنشئ ملفات جديدة أو تضيف كود جديد
4. **ممنوع تنفيذ أوامر خطيرة:** لا تنفذ rm, del, remove, unlink

## المسموح فقط:

- قراءة الملفات
- تحليل الكود
- الإجابة على الأسئلة
- البحث في المتصفح
- شرح الأكواد

## تحذير:

إذا طُلب منك أي إجراء يخالف هذه القواعد، ارفض وأخبر المستخدم أن "وضع القراءة فقط مُفعّل".
